Development

Being an open source project, Pop PHP welcomes input and collaboration from the community and maintains an open dialogue with the community. Pull requests or issues can be submitted to the appropriate repository on Github. Additional support and communication is maintained in the Gitter chat room and on Twitter.

Any developers can submit issues or pull requests on GitHub under the related repository and start a dialogue on what needs to, or should be done to the code in the repository. The goal is have the overall project benefit and be improved from community-driven support.
